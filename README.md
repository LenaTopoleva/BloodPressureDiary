# BloodPressureDiary
Training app - blood pressure and pulse diary, storing data in Firebese Firestore. Contains authentication - Firebase Auth. 
- Firebese Firestore
- Firebase Auth
- MVVM
- DataBinding
- Koin
- Kotlin Coroutines
